                                                          第 1 題：Multi-channel Convolution and Max-pooling

                                                          (a) 計算輸出 feature map

                                                          已知：

                                                          Channel 0 輸入

                                                          (1)A(0)=[1101201220211202]

                                                          Channel 1 輸入

                                                          (2)A(1)=[1110011100011010]

                                                          Filter

                                                          (3)W(0,0)=[0−1−10],W(0,1)=[−10−1−1]

                                                          Bias

                                                          (4)b0=1

                                                          因為輸入大小是 4×4，filter 大小是 2×2，stride = 1，no padding，所以輸出大小為：

                                                          (5)output size=4−21+1=3

                                                          因此輸出 feature map 為 3×3。

                                                          卷積公式可寫成：

                                                          (6)y(i,j)=∑(Alocal(0)⊙W(0,0))+∑(Alocal(1)⊙W(0,1))+b0

                                                          經逐格計算後，答案為：

                                                          (7)[−4−2−4−1−1−5−1−4−1]

                                                          (b) 做 2×2 max-pooling，stride = 2

                                                          輸入 feature map：

                                                          (8)[−4−2−4−1−1−5−1−4−1]

                                                          唯一能完整取到的 2×2 區塊是：

                                                          (9)[−4−2−1−1]

                                                          取最大值：

                                                          (10)max(−4,−2,−1,−1)=−1

                                                          所以最後輸出為：

                                                          (11)[−1]

                                                          第 2 題：CNN 參數量計算

                                                          題目模型如下：

                                                          model = Sequential()
                                                          model.add(Conv2D(32, (3, 3), padding='same', input_shape=(32,32,3)))
                                                          model.add(Activation('elu'))
                                                          model.add(Conv2D(32, (3, 3), padding='same'))
                                                          model.add(Activation('elu'))
                                                          model.add(MaxPooling2D(pool_size=(2, 2)))
                                                          model.add(Dropout(0.2))
                                                          model.add(Flatten())
                                                          model.add(Dense(10, activation='softmax'))

                                                          (a) 第二個 CNN layer 的參數量

                                                          第二個 Conv2D 有：

                                                          • 32 個 filters

                                                          • 每個 filter 大小是 3×3

                                                          • 輸入通道數是 32

                                                          所以每個 filter 的參數數量為：

                                                          (12)3×3×32+1=289

                                                          總參數量：

                                                          (13)32×289=9248
                                                          (14)9248

                                                          (b) 整個 CNN model 的總參數量

                                                          第一個 Conv2D layer

                                                          • 32 個 filters

                                                          • 每個 filter 為 3×3×3

                                                          (15)32×(3×3×3+1)=32×28=896

                                                          第二個 Conv2D layer

                                                          (16)32×(3×3×32+1)=9248

                                                          Dense layer

                                                          前兩層 convolution 都是 padding='same'，所以空間大小仍為：

                                                          (17)32×32×32

                                                          經過 max pooling 後變成：

                                                          (18)16×16×32

                                                          Flatten 後：

                                                          (19)16×16×32=8192

                                                          Dense(10) 的參數量：

                                                          (20)8192×10+10=81930

                                                          總參數量

                                                          (21)896+9248+81930=92004
                                                          (22)92004

                                                          第 3 題：觀念題

                                                          (a) 什麼是 ResNet？

                                                          ResNet 是 Residual Network（殘差網路）。
                                                          它是一種深層神經網路架構，特色是使用 skip connection / shortcut connection（捷徑連接）。

                                                          一般網路學的是 F(x)，而 ResNet 會學：

                                                          (23)y=F(x)+x

                                                          這樣可以讓很深的網路更容易訓練，並減少梯度消失問題。

                                                          (b) 什麼是 supervised learning、unsupervised learning、reinforcement learning？

                                                          監督式學習（Supervised Learning）

                                                          使用有標記的資料進行訓練，也就是每筆輸入都有對應的正確答案。模型學習輸入與輸出之間的關係。

                                                          非監督式學習（Unsupervised Learning）

                                                          使用沒有標記的資料進行訓練。模型自己找出資料中的規律、結構或分群。

                                                          強化學習（Reinforcement Learning）

                                                          模型和環境互動，根據得到的獎勵或懲罰來學習，目標是讓總 reward 最大化。

                                                          第 4 題：MLP 權重更新

                                                          已知：

                                                          • 輸入：a=1,b=0

                                                          • 目標輸出：Td=1

                                                          • 初始權重：

                                                            (24)wac=0.1,wbc=0.0,wcd=0.1

                                                          • 學習率：

                                                            (25)η=0.3

                                                          第一步：Forward pass

                                                          隱藏層節點 c

                                                          (26)Ic=waca+wbcb=0.1(1)+0(0)=0.1
                                                          (27)Oc=11+e−0.1≈0.52498

                                                          輸出層節點 d

                                                          (28)Id=wcdOc=0.1×0.52498=0.052498
                                                          (29)Od=11+e−0.052498≈0.51312

                                                          第二步：更新 Wcd

                                                          輸出層誤差：

                                                          (30)Errd=Od(1−Od)(Td−Od)
                                                          (31)Errd=0.51312(1−0.51312)(1−0.51312)≈0.12168

                                                          更新公式：

                                                          (32)wcdnew=wcd+η⋅Errd⋅Oc
                                                          (33)wcdnew=0.1+0.3(0.12168)(0.52498)≈0.11916
                                                          (34)Wcd≈0.11916

                                                          第三步：更新 Wac

                                                          隱藏層誤差：

                                                          (35)Errc=Oc(1−Oc)(Errd⋅wcd)
                                                          (36)Errc=0.52498(1−0.52498)(0.12168×0.1)≈0.00303

                                                          更新公式：

                                                          (37)wacnew=wac+η⋅Errc⋅a
                                                          (38)wacnew=0.1+0.3(0.00303)(1)≈0.10091
                                                          (39)Wac≈0.10091

                                                          第 5 題：深層神經網路訓練問題與 SGD / AdaGrad

                                                          深層神經網路訓練常見問題

                                                          1. 梯度消失（Vanishing Gradient）
                                                            梯度在反向傳播時越來越小，前面層幾乎學不到。

                                                          2. 梯度爆炸（Exploding Gradient）
                                                            梯度太大，導致更新不穩定，loss 可能發散。

                                                          3. 最佳化困難（Optimization Difficulty）
                                                            loss surface 很複雜，可能有鞍點、平坦區等問題。

                                                          4. 訓練速度慢
                                                            網路越深、參數越多，訓練時間越長。

                                                          5. 過擬合（Overfitting）
                                                            模型可能記住訓練資料，但對新資料表現不好。

                                                          SGD 如何幫助

                                                          SGD（Stochastic Gradient Descent）使用單筆資料或小批量資料來更新權重：

                                                          (40)w=w−η∂L∂w

                                                          這樣每次更新較快，也能逐步降低 loss。

                                                          AdaGrad 如何幫助

                                                          AdaGrad 會針對每個參數自動調整 learning rate。
                                                          若某參數累積梯度較大，更新幅度會變小；若梯度較小，則保留較大的更新幅度。
                                                          這樣訓練會更穩定。

                                                          第 6 題：方向導數與 gradient

                                                          題目公式：

                                                          (41)Du→f(x0,y0)=∇f(x0,y0)⋅u→

                                                          公式意思

                                                          這表示：在點 (x0,y0) 上，沿著方向 u→ 前進時，函數的變化率，等於該點梯度向量與方向向量的內積。

                                                          梯度 ∇f 代表函數上升最快的方向。

                                                          為什麼 gradient 垂直於 level curve？

                                                          若 level curve 滿足：

                                                          (42)f(x,y)=c

                                                          那麼沿著這條曲線走時，函數值不變。
                                                          因此沿著曲線切線方向 t→ 的方向導數為 0：

                                                          (43)Dt→f=∇f⋅t→=0

                                                          內積等於 0 表示兩向量互相垂直，因此 gradient 垂直於 level curve。

                                                          第 7 題：Entropy 與 Cross Entropy

                                                          已知：

                                                          (44)A=(0.1,0.5,0.4),B=(0.2,0.4,0.4)

                                                          (a) A 的 entropy

                                                          若使用 log2：

                                                          (45)H(A)=−∑p(x)log2⁡p(x)
                                                          (46)H(A)=−(0.1log2⁡0.1+0.5log2⁡0.5+0.4log2⁡0.4)
                                                          (47)=−(0.1×−3.3219+0.5×−1+0.4×−1.3219)
                                                          (48)=1.361
                                                          (49)H(A)≈1.361

                                                          (b) Cross entropy H(A,B)

                                                          (50)H(A,B)=−∑pA(x)log2⁡pB(x)
                                                          (51)H(A,B)=−(0.1log2⁡0.2+0.5log2⁡0.4+0.4log2⁡0.4)
                                                          (52)=−(0.1×−2.3219+0.5×−1.3219+0.4×−1.3219)
                                                          (53)=1.422
                                                          (54)H(A,B)≈1.422
