                                                            第 4 題：MLP 權重更新

                                                            已知

                                                            • a=1,b=0,Td=1

                                                            • wac=0.1,wbc=0,wcd=0.1

                                                            • η=0.3

                                                            Forward pass

                                                            (16)Ic=0.1(1)+0(0)=0.1
                                                            (17)Oc=11+e−0.1≈0.52498
                                                            (18)Id=0.1×0.52498=0.052498
                                                            (19)Od=11+e−0.052498≈0.51312

                                                            Output error

                                                            (20)Errd=Od(1−Od)(Td−Od)≈0.12168

                                                            更新 Wcd

                                                            (21)wcdnew=wcd+η⋅Errd⋅Oc
                                                            (22)wcdnew=0.1+0.3(0.12168)(0.52498)≈0.11916
                                                            (23)Wcd≈0.11916

                                                            Hidden error

                                                            (24)Errc=Oc(1−Oc)(Errd⋅wcd)≈0.00303

                                                            更新 Wac

                                                            (25)wacnew=wac+η⋅Errc⋅a
                                                            (26)wacnew=0.1+0.3(0.00303)(1)≈0.10091
                                                            (27)Wac≈0.10091

                                                            第 6 題：方向導數與 gradient

                                                            公式

                                                            (28)Du→f(x0,y0)=∇f(x0,y0)⋅u→

                                                            背法

                                                            • 方向導數 = 某點沿某方向的變化率

                                                            • gradient = 上升最快方向

                                                            為什麼 gradient 垂直 level curve？

                                                            因為 level curve 上函數值不變，所以沿切線方向的方向導數為 0：

                                                            (29)∇f⋅t→=0

                                                            內積為 0 代表垂直，所以 gradient 垂直 level curve。
